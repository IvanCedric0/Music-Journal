## ğŸ“¦ Installation (Local)

1. Clone this repo  
   `git clone https://github.com/IvanCedric0/Music-Journal.git`

2. Install dependencies  
   `npm install`

3. Set up `.env` file:
   ```env
   PG_USER=your_user
   PG_HOST=localhost
   PG_DATABASE=music_journal
   PG_PASSWORD=your_password
   PG_PORT=5432
   ```

4. Run the server  
   `npm start`
